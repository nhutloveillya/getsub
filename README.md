# GetSub

Một shell script được tạo ra để download sub của một season anime trong repo [boydaihungst/AnimeSubtitles](https://git.linuxholic.com/boydaihungst/AnimeSubtitles). Vì tôi không muốn tải tất cả vì quá nặng cũng như quá lười để tải từng file sub một.


# Cách sử dụng

Có thể sử dụng trực tiếp bằng cách chạy lệnh:
- Nếu bạn sài bash:
```
bash <(curl -sSL https://raw.githubusercontent.com/nhutloveillya/getsub/main/get.sh)
```
- Với fish:
 ```
bash (curl -sSL https://raw.githubusercontent.com/nhutloveillya/getsub/main/get.sh | psub)
 ```
